# Package "brmath"
Base functions for analytical solutions of geohydrological problems as defined in: *Analytical solutions of geohydrological problems* by G.A. Bruggeman.

## Installation

You can install the released version of brmath from with:

`install_github("KeesVanImmerzeel/brmath")`

Then load the package with:

`library("brmath")` 


# References

1. *Analytical solutions of geohydrological problems*
G.A. Bruggeman.
